## 1. 출력 구조

```


output/
├── en/raw/영상.raw.srt   ← Whisper 전사 원본 (재사용됨)
├── en/영상.en.srt        ← 후처리된 영어 SRT (번역 입력)
└── ko/영상.ko.srt        ← 한국어 번역 SRT (최종)


```

| 폴더 | 내용 | 재생성 조건 |
|------|------|-------------|
| `en/raw/` | Whisper 원본 그대로 (필러, 긴 블록 포함) | 삭제 시에만 재전사 |
| `en/` | 후처리 완료 (분할, 필러제거, 병합) | raw 기반으로 생성 |
| `ko/` | GPT-4o 번역 결과 | en.srt 기반으로 생성 |

## 2. 처리 파이프라인

```
영상 → [1] Whisper 전사 → [2] 영어 후처리 → [3] GPT-4o 번역
         (raw.srt)           (en.srt)          (ko.srt)
```

### [1단계] 전사 (Whisper API)
- 영상에서 오디오 추출 → 25MB 초과 시 자동 분할
- 결과를 `en/raw/`에 저장, 한 번 생성되면 재사용

### [2단계] 영어 후처리
1. 8초 초과 블록 → 문장 단위 분할
2. 필러 제거 ("Uh.", "Um.")
3. 불완전 문장 병합 (종속절, 관계절, 시간갭 감지)
4. 화자 전환 분리 (질문→답변, 응답어 감지)

### [3단계] 번역 (GPT-4o)
- 30블록씩 청크 + 앞뒤 5블록 문맥 제공
- 12가지 번역 규칙 (기술용어 영문유지, 경어체, 부호구조 유지 등)
- 실패 시 자동 재시도 → 최종 실패 시 원문 유지

---

## 3. 발견된 문제와 개선

### 문제: 한 문장이 여러 블록으로 쪼개짐

```
471: "because I'm querying for title and name."
472: "Uh, this line 11 essentially."
473: "Isn't doing anything aside from one important fact, which is."
```
→ 실제로는 하나의 문장. 각각 번역되면 부자연스러움.

### 개선: 5가지 휴리스틱으로 자동 병합

| 휴리스틱 | 설명 |
|----------|------|
| 종속절 감지 | `because`, `since` 등으로 시작 → 병합 |
| 관계절 감지 | `which is`, `that is`로 끝남 → 병합 |
| 시간 갭 감지 | 블록 간 0.5초 미만 + 소문자 시작 → 병합 |
| 필러 감지 | `uh`, `um`으로 시작 → 병합 |
| 화자 전환 | `Yeah`, `No` 등 응답어 → 분할 |

## 4. 번역 안정성

```
1단계: 전체 청크(30블록) 번역 시도
  ↓ 실패 시
2단계: 누락된 id만 재요청
  ↓ 실패 시
3단계: 원문(영어) 유지
```

- JSON Schema strict 모드 (id 매핑 보장)
- 용어집(`glossary.json`) 자동 적용
- 기술 용어(Anzo, SPARQL, RDF 등) 영문 유지

## 5. 현재 상태

영상 전사(Whisper)와 영어 후처리까지는 정상 완료되었으나, 번역 단계에서 OpenAI API 크레딧이 소진되어 한국어 번역이 실행되지 않는 상태입니다. 크레딧 충전 후 재실행하면 이미 전사된 결과를 재사용하여 번역만 수행하므로 추가 전사 비용은 발생하지 않습니다.

## 6. 확인 방법

**공유폴더 경로**: `\\intranet.spss.co.kr\개인폴더\권체은\SRTLinker`

### 폴더 진입 후 확인 사항

1. `output\en\raw\` → Whisper 전사 원본 확인 (`.raw.srt`)
2. `output\en\` → 후처리된 영어 자막 확인 (`.en.srt`)
3. `output\ko\` → 한국어 번역 결과 확인 (`.ko.srt`)

### 실행 방법 (크레딧 충전 후)

1. `SRTLinker.bat` 더블클릭 → 브라우저 자동 열림
2. 영상 파일 드래그 또는 "파일 선택" 클릭
3. "번역 시작" 클릭
4. 완료 후 `output\ko\` 폴더에서 `.ko.srt` 파일 확인

### 필요 조치

- OpenAI 크레딧 충전 (https://platform.openai.com/account/billing)
- 충전 완료 후 재실행 시 전사 과정 생략, **번역만 수행** (1시간 영상 기준 3~5분)
