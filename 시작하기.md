# SRTLinker 사용 가이드

영상/오디오 파일을 자동으로 전사하고 한국어로 번역하는 프로그램입니다.

## 📋 사전 준비

**Python 3.10 이상** 설치
- 다운로드: https://www.python.org/downloads/
- ⚠️ 설치 시 **"Add Python to PATH"** 반드시 체크!

## 🚀 설치 및 실행 (2분)

### 1단계: 설치
`install.bat` 파일을 **더블클릭**하세요.
- 자동으로 필요한 패키지가 설치됩니다 (1~2분 소요)
- OpenAI API 키는 이미 설정되어 있습니다

### 2단계: 실행
`SRTLinker.bat` 파일을 **더블클릭**하세요.
- 브라우저가 자동으로 열립니다

## 💡 사용 방법

1. 영상/오디오 파일을 **드래그앤드롭**
2. **번역 시작** 버튼 클릭
3. 완료되면 `output` 폴더에서 결과 확인
   - `output/en/` - 영어 자막
   - `output/ko/` - 한국어 자막

## 📁 지원 파일 형식

- **영상**: mp4, mkv, mov, avi, webm 등
- **오디오**: mp3, wav, m4a, aac 등
- **자막**: srt (번역만 하는 경우)

## ❓ 문제 해결

### Python이 설치되지 않았다고 나올 때
- Python 재설치 시 "Add Python to PATH" 체크 확인
- 명령 프롬프트에서 `python --version` 입력해서 확인
